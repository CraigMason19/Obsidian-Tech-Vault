## Local Hosting

[LM Studio](https://lmstudio.ai) 
- Can be used to run LLM's locally.

[Continue ](https://marketplace.visualstudio.com/items?itemName=Continue.continue)
- Visual Studio Extension
- AI Code Agent that allows you to talk to downloaded and configured LLM'S

[YT - Indently - Build a Local AI Agent in Python in only 15 Minutes](https://www.youtube.com/watch?v=LykXu60aKoY)
[YT -  The Best Local Agentic Coding Workflow (Complete Guide)](https://www.youtube.com/watch?v=UngVdAsQEiU&list=WL&index=44&t=14s)


LLM'S can be hosted locally (on your machine) allowing to easily swap models and make use of them offline. This does however require a good CPU as it will work off the V-RAM
