## Snap measure

| Combo                                   | Effect                |
| --------------------------------------- | --------------------- |
| CTRL + SHIFT + V                        | Enter measure mode    |
| (In mode)                               | ALT to change lines   |
| (In mode) + Hover over any midpoint + Q | Snap measure pie menu |
